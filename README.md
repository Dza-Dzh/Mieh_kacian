# Management Parking Mieh Kacian

Sistem ini adalah aplikasi **manajemen parkir**.  
Fokusnya pada pencatatan kendaraan, pemantauan data parkir, pencarian cepat, statistik, dan riwayat parkir.

## ✨ Fitur 

1. **Data Kendaraan Masuk**
	- ID Ticket
	- Nomor Plat
	- Jenis Kendaraan
	- Jam Masuk Kendaraan

2. **Menampilkan Daftar Kendaraan yang Sedang Terparkir**

3. **Update/Ubah Data Kendaraan yang Sedang Terparkir**

4. **Data Kendaraan Keluar**
	- Nomor Plat
	- Jam Keluar
	- Harga

5. **Cari Kendaraan**
	- Nomor Plat
	- Jam Masuk
	- ID Ticket

6. **Statistik Parkir** (Harian, Mingguan, Bulanan, Tahunan)
	- Total Masuk
	- Total Keluar
	- Total Pendapatan
	- Total Mobil
	- Total Motor
	- Total Kendaraan

7. **Riwayat Parkir**
	- Nomor Plat
	- Jenis Kendaraan
	- ID Ticket
	- Waktu Masuk
	- Waktu Keluar
	- Status (parkir/out)
	- Biaya Parkir
